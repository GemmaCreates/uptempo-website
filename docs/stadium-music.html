What it does

Stadium Music: Walkout Tracks makes it easy to create a team, add players, assign walkout songs, and run a clean, tap-to-play game day experience. Parents can submit song requests via a simple web form using Team Link, and you can approve submissions inside the app.

COACHES Run walkout music in seconds
  • Create your team and roster
  • Assign songs via Apple Music or local files
  • Set clip start time + duration per player
  • Use Gameday Mode for fast playback
PARENTS Submit songs without an app
  • Open the Team Link on any phone
  • Paste an Apple Music link to the song
  • Add optional start time + notes
  • Coach reviews and approves

Key features

Team Link UNIQUE

Generate a shareable URL or QR code so parents can submit walkout song requests. No login required.

Apple Music integration

Search and play songs from Apple Music using MusicKit. Playback typically requires an active Apple Music subscription.

Local file support

Import MP3/M4A files from the Files app for custom audio, local artists, or anything not on Apple Music.

Gameday mode

Big, tappable controls designed for outdoor visibility and quick player introductions.

How it works

1

Create team

Set up your team and add players with jersey numbers. Customize team colors and season info.

2

Share Team Link

Generate a unique link or QR code. Parents open it on any device to submit song requests with Apple Music URLs.

3

Approve + play in Gameday Mode

Review submissions in the app, approve songs, set clip times, then use Gameday Mode for fast, one-tap playback.

Requirements

  • iOS 16.0+
  • Internet connection for Apple Music search/streaming and Team Link submissions
  • Apple Music subscription for full Apple Music playback
